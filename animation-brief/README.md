## How to Use This

1. Capture the 8 screenshots (see `screenshots/CAPTURE-GUIDE.md`)
2. Open Replit Animation
3. Upload all 8 screenshots
4. Copy the prompt from `01-replit-prompt.md`
5. Click **Enhance Prompt** before submitting
6. After first render, use the follow-up prompts in `01-replit-prompt.md`
